# VI-BCI: Visual Imagery EEG Decoding

Decoding of visual imagery category (Living / Geometric / Object) from scalp EEG using the [VI-BCI dataset](https://doi.org/10.6084/m9.figshare.30227503) (Figshare, 2026). Pipeline: preprocessing → discovery visualizations → within-subject decoding → cross-subject decoding (optional, via Kaggle) → VVIQ correlation. This repository provides the code and run order for reproducibility.

---

## Dataset

- **VI-BCI**, 22 participants, 32 channels, 1000 Hz.  
- **Download:** [Figshare DOI 10.6084/m9.figshare.30227503](https://doi.org/10.6084/m9.figshare.30227503).  
- **Place data so that:** raw BDF and events TSV live under `dataset/subjects/` (see dataset README for exact layout).  
- **Preprocessed outputs** (`.fif` epochs) are generated by the first script and saved under `outputs/preprocessed/`.

## Pipeline (run order)

| Step | Command | Output |
|------|--------|--------|
| 1 | `python preprocess_all.py full` | `outputs/preprocessed/*.fif` |
| 2 | `python visualize_discovery.py` | `outputs/figures/` (ERPs, topomaps, TFR), `outputs/logs/phase3_findings.txt` |
| 3 | `python within_subject_decoder.py` | `outputs/logs/within_subject_results.txt` |
| 4 | `python vviq_correlation.py` | `outputs/figures/vviq_scatter.png` |
| 5 | Cross-subject (Kaggle) | Upload preprocessed `.fif` as a Kaggle dataset; run `colab_pipeline.ipynb` on Kaggle (GPU). See [KAGGLE_SETUP.md](KAGGLE_SETUP.md). Outputs: `vibci_outputs/` (models, logs, figures). |

**Scripts (one-line description):**

- **preprocess_all.py** — Load raw BDF, filter, ICA, epoch, reject bad trials; write `.fif` epochs.
- **visualize_discovery.py** — ERP comparison, topomaps, time–frequency, channel importance (Living/Geometric/Object).
- **within_subject_decoder.py** — Train on session 1, test on session 2; reports 10- and 3-class accuracy per subject.
- **train_decoder.py** — Cross-subject LOSO decoder (Transformer); use `--ea` for Euclidean Alignment. Heavy on CPU; use Colab for full run.
- **vviq_correlation.py** — Pearson correlation and median-split (High/Low VVIQ) vs 3-class accuracy; scatter plot.
- **align_subject.py** — Euclidean Alignment helpers (used by decoder scripts and Colab).
- **colab_pipeline.ipynb** — Cross-subject LOSO pipeline for **Kaggle** (GPU, more RAM); reads `.fif` from an input dataset, writes to `/kaggle/working/vibci_outputs/`. See [KAGGLE_SETUP.md](KAGGLE_SETUP.md).

## Output figures

Generated figures live in **`outputs/figures/`**. 

| File | Description |
|------|-------------|
| `outputs/figures/erp_comparison.png` | Grand-average ERPs for Living / Geometric / Object on O1, Oz, O2, T7, T8; significance shading on Oz. |
| `outputs/figures/topomaps_supercategories.png` | Scalp topomaps at 100, 200, 500, 1000 ms for the three supercategories. |
| `outputs/figures/tfr_alpha_beta.png` | Time–frequency (alpha 8–12 Hz, beta 13–30 Hz) for Living vs Geometric over occipital channels. |
| `outputs/figures/channel_importance.png` | Mean amplitude 0–2 s post-stimulus per channel for each supercategory. |
| `outputs/figures/vviq_scatter.png` | VVIQ score vs 3-class accuracy; regression line; chance line; subject labels. |
| `outputs/figures/confusion_matrix_10class.png` | Combined LOSO confusion matrix (10 categories). |
| `outputs/figures/confusion_matrix_3class.png` | Combined LOSO confusion matrix (3 supercategories). |
| `outputs/figures/per_subject_accuracy.png` | Per-subject 10- and 3-class accuracy bar chart (LOSO). |
| `outputs/figures/attention_topomap.png` | Channel importance from best cross-subject fold (spatial attention). |
